# Guru Blog Platform

This repository contains both the frontend and backend for the blog platform.

## Directory Structure

- `frontend/`: Contains the React frontend application.
- `backend/`: Contains the Node.js backend application.

## Setup Instructions

### Frontend

```sh
cd frontend
npm install
npm start

### Backend


cd backend
npm install
npm start
```
